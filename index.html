Javascript Phone Number Challenge

Ok, here's the challenge

Below is an input box. In that input, a user types in a phone number.

Upon selecting [submit], the program should show the 'formatted' phone number


Examples

For example:

6172324567 should produce : 1 (617)-232-4567

19176768311 should produce : 1 (917)-676-8311


Don't forget international numbers:

176172324567 should produce : 17 (617)-232-4567

39176768311 should produce : 3 (917)-676-8311


As a developer, these types of problems are our bread and butter. The primary goal of this challenge is for you to learn the rudimentary basics of javascript in the hopes that it will allow you to better relate and communicate to developers. Good luck.
